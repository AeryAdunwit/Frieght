# Knowledge Sheets Template

ใช้เอกสารนี้เป็นแม่แบบสำหรับเติมข้อมูลลง Google Sheets ก่อน sync เข้า Supabase

## Required Columns
- `question`
- `answer`

## Optional Columns
- `keywords`
- `active`

ค่าที่แนะนำ:
- `active = yes`
- `keywords` ใส่คำพ้อง, คำค้น, ชื่อเรียกที่ลูกค้ามักใช้

## Sheet Tabs Recommended
- `solar`
- `pricing`
- `booking`
- `claim`
- `general`
- `coverage`
- `documents`
- `timeline`

## Writing Guidance
- คำถาม 1 แถว = 1 เจตนา
- คำตอบควรใช้ภาษาที่ลูกค้าอ่านแล้วเข้าใจง่าย
- ถ้ามีเงื่อนไข ให้เขียนเงื่อนไขและ next step ให้ครบ
- ใส่คำพ้องใน `keywords` เช่น `ราคา, quote, quotation, ค่าส่ง`
- ถ้าข้อมูลเลิกใช้แล้วให้ตั้ง `active = no`
